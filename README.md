Voice-Controlled ASCII Art Generator
カメラ映像から人物（輝度）を認識し、リアルタイムの音声認識結果で構成されるアスキーアート（AA）を生成するPythonスクリプトです。

機能
• リアルタイム音声認識: マイクに向かって話した言葉が即座にアスキーアートの構成文字になります。

• 動的アスキーアート: カメラ映像の明るさに合わせて、文字の密度や色（輝度）が変化します。

• 並列処理: 音声認識を別スレッドで実行するため、映像が止まることなくスムーズに表示されます。

必要ライブラリ
```

pip install opencv-python numpy SpeechRecognition PyAudio Pillow

```

使い方
1. スクリプトを実行します。

2. カメラが起動し、最初は「READY」という文字でAAが表示されます。

3. マイクに向かって日本語で話しかけると、その言葉にAAが書き換わります。

4. `ESC` キーを押すと終了します。

仕組み
1. 音声認識 (SpeechRecognition): Google Speech Recognition APIを使用して音声をテキスト化します。

2. 映像処理 (OpenCV): カメラ映像をグレースケール化し、リサイズして各ピクセルの輝度を取得します。

3. 描画 (Pillow): 黒いキャンバス上に、輝度が一定以上の場所に音声認識した文字を描画します。
